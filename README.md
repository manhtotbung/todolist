# EasyTasks - Quản lý công việc cá nhân

Easy Tasks là ứng dụng web giúp bạn quản lý các nhiệm vụ hàng ngày một cách đơn giản, trực quan và hiện đại. Dự án này gồm cả backend (Node.js/Express/MongoDB) và frontend (React + Vite + TailwindCSS), triển khai sẵn sàng cho môi trường production.

## Tính năng nổi bật
- Đăng ký, đăng nhập, đăng xuất tài khoản
- Thêm, sửa, xóa nhiệm vụ cá nhân
- Đánh dấu hoàn thành, lọc nhiệm vụ theo trạng thái (Tất cả, Đang làm, Hoàn thành)
- Lọc nhiệm vụ theo ngày (Hôm nay, Tuần này, Tháng này)
- Phân trang danh sách nhiệm vụ
- Giao diện responsive, dễ dùng trên cả máy tính và điện thoại
- Thông báo realtime (toaster) khi thao tác thành công/thất bại

## Công nghệ sử dụng
- **Frontend:** React, Vite, TailwindCSS, Zustand, React Hook Form, Zod, Sonner
- **Backend:** Node.js, Express, MongoDB, Mongoose, JWT, bcrypt
